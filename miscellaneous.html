Question:-What is CDS :-

Core data service is mainly a view if you are a ANAB developer then you have to write query, procedure and view in HANA database but after CDS as developer you don't need to go to database , you can write CDS view in application server and that will be converted into HANA view and procedure

Note:- SAP HANA has two different databases, System DB and Tenant DB. The system database is used for system management/administration. It contains the landscape information including the tenant database information.
Note:- SAP HANA supports multiple isolated databases in a single SAP HANA system. These databases are referred to as the tenant database. The tenant database is where all customer data resides.
