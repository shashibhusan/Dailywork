Important Docker command
1-docker build -t spring-boot-docker.jar      (where t stand for target)
2-docker history      Show the history of an image
3-docker images      List images
4-docker logs      Fetch the logs of a container
5-docker pull      Pull an image or a repository from a registry
6-docker run      Run a command in a new container
7-docker ps      To get the docker process running
8-docker restart      Restart one or more containers
9-docker stop      one or more running containers
10-docker kill      Kill one or more running containers
