Step to push your local work in remote
Approch -A

Step 1- 
		git init in your local folder
Step 2-
		Create a repo on remote
Step 3- 
		Add remote origin to your local repo with below command
		git remote add origin git@github.com:shashibhusan/SpringBootWorkspace.git
Step 4-
		pull the changes of master if any like ReadMe with below command
		git pull origin master --allow-unrelated-histories
Step 5-
		finally push
		git push origin master
